# 🎮 ことばポーカー

**ひらがなカードで単語を作って役を揃えよう！**

ことばポーカーは、ひらがなカードを使って単語を作りながら、ポーカーの役も狙う新しいタイプのワードゲームです。単語の知識と戦略的思考が求められる、楽しくて教育的なゲームです。

## 📋 ゲーム概要

- **プレイ人数**: 1人（現在はソロプレイ対応）
- **プレイ時間**: 約10-15分
- **対象年齢**: 小学生以上
- **ジャンル**: ワードゲーム × カードゲーム

## 🎯 ゲームの目的

3ラウンドを通して、**ひらがなカードから意味のある単語を作り**、同時に**ポーカーの役**を作って高得点を目指します。最終的に最も多くの得点を獲得したプレイヤーの勝利です。

## 🃏 ゲーム準備

### カード構成
- **ひらがな47文字** × **4枚ずつ** = **合計188枚**
- 「あ」「い」「う」...「ん」まで、各文字が4枚ずつ入っています

### ゲームの流れ
1. 各ラウンド開始時に **5枚のカード** を配布
2. **カード交換**（1ラウンドに1回まで）
3. **単語作成** と **役判定**
4. **得点計算**
5. 次のラウンドへ（全3ラウンド）

## 🎲 詳しいルール

### 1. 手札の管理
- 毎ラウンド開始時に5枚のカードが配られます
- カードをクリックして選択し、単語を作成します
- 選択したカードは**ドラッグ&ドロップで順番を変更**できます

### 2. カード交換
- **1ラウンドに1回まで**、選択したカードを新しいカードと交換できます
- 交換したいカードを選択して「選択したカードを交換」ボタンをクリック
- 交換後は、そのラウンドでは再度交換できません

### 3. 単語作成
- 選択したカードで**意味のある日本語単語**を作成します
- **2文字以上**の単語が必要です
- **ひらがなのみ**で構成される必要があります

#### 有効な単語の例
- **動物**: ねこ、いぬ、たぬき、うさぎ
- **食べ物**: りんご、おにぎり、らーめん、すし
- **自然**: やま、うみ、そら、はな
- **乗り物**: でんしゃ、ひこうき、じてんしゃ
- **建物**: いえ、がっこう、びょういん

### 4. ゲームマスター機能 🎮
辞書に登録されていない単語でも、ゲームマスター（GM）が承認できます。

#### 使い方
1. 右上の「🎮 GM」ボタンをクリックしてGMモードを有効化
2. 未登録の単語を提出すると承認ダイアログが表示
3. 「✅ 承認」または「❌ 却下」を選択
4. 承認された単語は🎮マーク付きで表示されます

## 🏆 得点システム

### 基本得点
| 項目 | 得点 | 説明 |
|------|------|------|
| **単語作成** | +3pt | 有効な単語を作成 |
| **文字数ボーナス** | +1pt/文字 | 4文字以上の単語（4文字=+1pt、5文字=+2pt...） |
| **「ん」ボーナス** | +1pt | 単語に「ん」が含まれている場合 |

### ポーカー役ボーナス
同じ文字を複数使った場合の追加ボーナス

| 役名 | 得点 | 説明 | 例 |
|------|------|------|-----|
| **ワンペア** | +2pt | 同じ文字が2枚 | 「ねこ」(「ね」「こ」) |
| **ツーペア** | +4pt | 同じ文字のペアが2組 | 「ききき」など |
| **3カード** | +5pt | 同じ文字が3枚 | 「かかか」など |
| **フルハウス** | +8pt | 3枚+2枚の組み合わせ | 「ああああい」など |
| **4カード** | +10pt | 同じ文字が4枚 | 「あああああ」など |

### 得点計算例
**「りんご」（4文字）を作成した場合:**
- 単語作成: +3pt
- 4文字ボーナス: +1pt
- 「ん」ボーナス: +1pt
- **合計: 5pt**

**「ねこ」（ワンペア無し）を作成した場合:**
- 単語作成: +3pt
- **合計: 3pt**

## 🎮 操作方法

### 基本操作
1. **カード選択**: 手札のカードをクリック
2. **順序変更**: 選択エリアでカードをドラッグ&ドロップ
3. **カード削除**: 選択エリアのカードをクリック
4. **全選択解除**: 「選択クリア」ボタン

### ボタン説明
- **選択したカードを交換**: 選択カードを新しいカードと交換（1回限り）
- **選択クリア**: 全ての選択を解除
- **単語を提出**: 作成した単語を提出して得点計算
- **パス**: 単語を作らずにラウンドを終了（0pt）

## 🏁 ゲーム終了と勝利条件

### ゲーム終了
- **3ラウンド**でゲーム終了
- 各ラウンドの得点を合計して最終得点を決定

### 評価基準
| 最終得点 | 評価 |
|----------|------|
| **30点以上** | 素晴らしい！🎉 |
| **20点以上** | 良い成績です！👏 |
| **10点以上** | まずまずです！😊 |
| **10点未満** | がんばりましょう！💪 |

## 💡 戦略のヒント

### 初心者向け
1. **短い単語から始める**: 「ねこ」「いぬ」など覚えやすい単語
2. **交換を活用**: 良い単語が作れない時は積極的に交換
3. **「ん」を狙う**: +1ptボーナスで得点アップ

### 上級者向け
1. **長い単語を狙う**: 4文字以上で文字数ボーナス獲得
2. **役を意識する**: 同じ文字を集めてポーカー役を狙う
3. **GMモード活用**: 珍しい単語でGM承認を狙う
